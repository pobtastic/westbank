51521: Make the game run faster
Used by the routine at Init_Title_Screen
This routine is called at the start of each phase, and makes it faster than than the previous by decreasing the wait time. This starts at $500 (slowest) and is decreased by $40 for each call, so things generally run faster.

51533 OR L
51534 JR NZ,51539 No, so move on
51536 LD HL,64 The game speed cannot be 0, it's impossible, so make it $40.
