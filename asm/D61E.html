D61E: Write Character And Door To Buffer.
Door frames 2 and 3 both contain a "section" of a game character. This is copied into a buffer by first drawing the game character fully to DOOR_BUFFER. Then the door frame is drawn over the top (in the correct position).
Draws the game character together with door frame 2.
CHARACTER_DOOR_FRAME_2 D61E CALL CHARACTER_ADDR On entry A points to a character index. The call to CHARACTER_ADDR translates this to the memory location of the character frame data (held in HL) and then calls DOOR_BUFFER_COPY.
D621 CALL DOOR_BUFFER_COPY
DOOR_FRAME_2_COPY D624 LD HL,$BCE8 Copies DOOR_FRAME_2 into the DOOR_BUFFER (+2 bytes ~ $EA92).
