C941: Make the game run faster
Used by the routine at Init_Title_Screen
This routine is called at the start of each phase, and makes it faster than than the previous by decreasing the wait time. This starts at $500 (slowest) and is decreased by $40 for each call, so things generally run faster.
C941 PUSH AF Remember these
C942 PUSH HL
C943 LD HL,($CE1C) Get the current speed
C946 LD BC,$0040 Decrease it by this
C949 AND A Clear the carry flag so next instruction works
C94A SBC HL,BC Drop the speed
C94C LD A,H Is it zero?
C94D OR L
C94E JR NZ,$C953 No, so move on
C950 LD HL,$0040 The game speed cannot be 0, it's impossible, so make it $40.
C953 LD ($CE1C),HL Store it back
C956 POP HL Restore used registers
C957 POP AF
C958 RET
