CD7E: Debugger
As the game uses its own character set, it's unlikely this routine was/ is used for the actual game (it doesn't seem to be called from anywhere). At a guess, it was probably used for "testing" text messages and outputs during development.
Debugger CD7E LD A,(HL) Grab a byte from HL return if it's FF (our termination character).
CD7F CP $FF
CD81 RET Z
CD82 RST $10 Print to screen (see PRINT_A_1).
CD83 INC HL Increase HL by one and loop back to Debugger.
CD84 JR Debugger
